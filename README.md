# 🚀 Projeto DevAcademy - Sistema de Autenticação

Este é um projeto de interface de autenticação (Login e Cadastro) desenvolvido com foco em **UI/UX moderna**, utilizando práticas atuais de CSS e organização de código. O projeto faz parte do ecossistema de aprendizado da DevAcademy.

## 📋 Sobre o Projeto

O objetivo deste projeto é fornecer uma experiência de usuário fluida e responsiva para sistemas de gerenciamento de contas. A interface conta com:
* **Design Minimalista:** Uso da fonte 'Inter' e paleta de cores Slate/Indigo.
* **Feedback em Tempo Real:** Sistema visual para exibição de mensagens de erro ou sucesso.
* **Responsividade:** Adaptável para dispositivos móveis e desktops.
* **Validação de Formulários:** Campos estruturados para captura de dados e confirmação de senha.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica do projeto.
* **CSS3:** Estilização avançada com variáveis, Flexbox e efeitos de transição.
* **JavaScript (ES6+):** Lógica para manipulação de DOM, alternância de telas e validação.

---

## 📸 Demonstração da Interface

### Tela de Cadastro
A tela de cadastro utiliza um cartão centralizado com sombreamento suave e campos de input com estados de foco destacados.

> **Destaque:** O layout utiliza `box-shadow` e `border-radius: 20px` para um visual "clean" e profissional.
